# Alt+F+This

*"Office Space meets Universal Paperclips with duck-based resource allocation"*

A darkly comedic strategy game where you're a developer trapped in corporate hell, secretly building a startup on company time. Allocate your hours between soul-crushing job tasks and your escape plan. Ship buggy code to meet deadlines. Try not to get caught.

---

## Running the Game

1. Open the project in **Godot 4.5**
2. Press **F5**

That's it. Main scene is `scenes/game_ui.tscn`.
